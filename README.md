# Baseball Game⚾

**1. 요구사항**

   가) 숫자는 1~9까지 서로 다른 수로 이루어진 3자리수

   나) 힌트 : 

             * 같은 자리 + 같은 숫자 = 스트라이크      
             * 다른 자리 + 같은 숫자 = 볼
             * 다른 자리 + 다른 숫자 = 낫싱

   다) 컴퓨터가 상대방 역할을 하여 힌트에 따라 결과를 출력

   라) 3자리 숫자 모두를 맞추게 되면, 게임이 종료된다.
